# vynscastingmod :3 

An open source (GPL v3), feature rich, intuitive gorilla tag casting mod. 

Formerly known as frapster's casting mod pre-recode.

This casting mod is being made for the sole reason that there are no longer any good free casting mods. Casting should be free and open, not behind a paywall, sakuraa.

## Features:
- Doesn't work in VR.
- Adjustable camera rotational and movement smoothing
- Adjustable player smoothing (rig lerping)
- Adjustable camera FOV
- Adjustable offset via WASD
- GUI Room joiner
- Consistent smoothing, not based on FPS
- Head/Body tracking & first person casting
- Mic push-to-talk
- Non-bloated UI, control everything with hotkeys.
